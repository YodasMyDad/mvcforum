The Post property of MVCForumAPI exposes functionality associated with forum posts.

Members

Create

Functionality

Adds a new post to the system.

Parameters

"post"

Type: Post

Usage: the post to be added

Returns

Example

var newPost = new Post 
{
    PostContent = postContent,
    User = user,
    Topic = topic
};

_api.Post.Create(newPost);

Delete

Functionality

Deletes a post from the system.

Parameters

"post"

Type: Post

Usage: the post to be deleted.

Returns

Example

public void DeletePost(Post post)
{
     _api.Post.Delete(post);
}

GetPostsByMember

Functionality

Retrieves all the posts made by the specified member.

Parameters

"memberId"

Type: Guid

Usage: the identifier of the member.

Returns

A list of posts.

Example

public IList<Post> GetPosts(MembershipUser user)
{
    return _api.Post.GetPostsByMember(user.Id);
}

GetPostsByTopicId

Functionality

Retrieves all the posts within a topic.

Parameters

"topicId"

Type: Guid

Usage: the Id of the topic.

Returns

A list of topics.

Example

public IList<Post> GetPostsForTopic(Topic topic)
{
    return _api.Post.GetPostsByTopicId(topic.Id);
}

GetSolutionsByMember

Functionality

Retrieves all the posts made by a member of the forum that have been marked as a solution.

Parameters

"memberId"

Type: Guid

Usage: the Id of the member to search for solutions

Returns

Example

public IList<Post> GetSolutionsByMember(Guid memberId)
{
    return _api.Post.GetSolutionsByMember(memberId);
}
