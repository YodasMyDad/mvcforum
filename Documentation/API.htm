The MVCForum functionality is exposed through the class named "MVCForumAPI". You can use an object of this type when you are writing your own code to extend MVCForum.

If you have downloaded the source for MVCForum then you will be able to use all the functionality within the system when you develop code.

If however you are developing plug-ins to MVCForum, such as badges or events, then you will only be able to access the MVC functionality via an instance of MVCForumAPI.
