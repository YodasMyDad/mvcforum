The Member property of MVCForumAPI exposes functionality associated with forum members.

Members

Create

Functionality

Create a new member.

Parameters

"member"

Type: MembershipUser

Usage: The member to create.

Returns

The new member. An id is generated as part of the creation process.

Example

public MembershipUser Create(string name)
{
    var newUser = new MembershipUser
                      {
                          UserName = name
                      };

    // Id is created by API
    return _api.Member.Create(newUser);
}

Delete

Functionality

Delete a member.

Parameters

"member"

Type: Member

Usage: The member to delete.

Returns

void.

Example

public void DeleteMember(MembershipUser user)
{
    _api.Member.Delete(user);
}
